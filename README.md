### **✅ HACK2HIRE Requirements - 6/6 Complete**

| # | Feature | Implementation |
| --- | --- | --- |
| 1 | **Data & Health Score** | 200 customers with health scoring based on NPS, MRR, tickets |
| 2 | **Natural Language Query** | Parses `region`, `health`, `NPS`, `MRR` from plain English |
| 3 | **Filtering** | One-click `Healthy`, `At Risk`, `Critical` filters |
| 4 | **Email Draft** | Auto-generates personalized outreach for at-risk customers |
| 5 | **Churn Prediction** | ML-based risk % with top contributing factors |
| 6 | **Customer Management** | Full CRUD with customer names, region, metrics |

### **🛠️ Tech Stack**
- **Backend:** Python, Flask, SQLAlchemy
- **Database:** SQLite with 200 pre-seeded customers
- **ML:** Scikit-learn for churn prediction
- **Frontend:** HTML, CSS, Vanilla JS
- **NLQ:** Custom parser for entity extraction
